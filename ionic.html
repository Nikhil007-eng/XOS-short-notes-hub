Ionic Equilibrium – JEE Notes

1. Acid-Base Theories

Arrhenius Concept

Acid: Dissociates in water → H⁺

Base: Dissociates in water → OH⁻

HA(aq) ⇌ H⁺(aq) + A⁻(aq)

Brønsted–Lowry Concept

Acid: Proton donor

Base: Proton acceptor

Lewis Concept

Acid: Electron pair acceptor (e.g., BF₃)

Base: Electron pair donor (e.g., NH₃)

2. Ionic Product of Water & pH

H₂O ⇌ H⁺ + OH⁻
K_w = [H⁺][OH⁻] = 10⁻¹⁴ (at 25°C)
pH = -log[H⁺]
pOH = -log[OH⁻]
pH + pOH = 14 (at 25°C)
Note: Neutral pH changes with temperature:
• T > 25°C → pH < 7 for neutral water
• T < 25°C → pH > 7 for neutral water

3. pH Calculations

Strong Acid/Base

[H⁺] = C (for monoprotic strong acid)
[OH⁻] = C (for monoprotic strong base)
For Ba(OH)₂: [OH⁻] = 2 × concentration

Weak Acid/Base

For weak acid HA: Ka = [H⁺][A⁻]/[HA]
If α ≤ 0.05: α = √(Ka/C)
[H⁺] = Cα = √(Ka × C)
pH = ½(pKa – log C)
For weak base BOH: Kb = [BH⁺][OH⁻]/[BOH]
[OH⁻] = √(Kb × C)
pOH = ½(pKb – log C)

Very Dilute Strong Acid

If [H⁺] from acid < 10⁻⁶ M, consider contribution from water.

4. Salt Hydrolysis

Salt Type Hydrolysis Constant (Kh) pH Formula
Weak Acid + Strong Base (e.g., CH₃COONa) Kh = Kw/Ka pH = 7 + ½ pKa + ½ log C
Strong Acid + Weak Base (e.g., NH₄Cl) Kh = Kw/Kb pH = 7 – ½ pKb – ½ log C
Weak Acid + Weak Base (e.g., CH₃COONH₄) Kh = Kw/(Ka·Kb) pH = 7 + ½(pKa – pKb)

5. Buffer Solutions

Acidic Buffer (Weak Acid + Salt of its conjugate base)

Henderson–Hasselbalch equation:
pH = pKa + log([Salt]/[Acid])

Basic Buffer (Weak Base + Salt of its conjugate acid)

pOH = pKb + log([Salt]/[Base])
pH = 14 – pOH
Buffer capacity is maximum when [Salt] = [Acid/Base]

6. Solubility Product (Ksp)

Salt Type Ionization Ksp Expression
AB (e.g., AgCl) AB ⇌ A⁺ + B⁻ Ksp = S²
AB₂ or A₂B (e.g., CaF₂) AB₂ ⇌ A²⁺ + 2B⁻ Ksp = 4S³
A₂B₃ (e.g., Bi₂S₃) A₂B₃ ⇌ 2A³⁺ + 3B²⁻ Ksp = 108S⁵
AxBy ⇌ xA + yB Ksp = (xS)x(yS)y
General: S = [Ksp/(xx·yy)]1/(x+y)

Common Ion Effect

Solubility decreases in presence of common ion.

For AgCl in NaCl solution:
Ksp = [Ag⁺][Cl⁻] = S(S + [Cl⁻]added)

7. Indicators

HIn ⇌ H⁺ + In⁻
pH = pKIn + log([In⁻]/[HIn])

Color change range: pH = pKIn ± 1

Key Points to Remember

  • For weak acids/bases, use approximation only if α ≤ 0.05.
  • In very dilute acids ([H⁺] < 10⁻⁶ M), consider water’s contribution.
  • Buffer works best when pH ≈ pKa (for acidic buffer).
  • Salt hydrolysis depends on relative strengths of acid and base.
  • Selective precipitation: salt with lowest [ion] needed to precipitate first.
  • Solubility in common ion solution < solubility in pure water.
